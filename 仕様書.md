# Area Captcha Tool 仕様書

## 1. プロジェクト概要
本プロジェクト（area-captcha）は、Windows環境において指定した画面領域を連続してスクリーンショット撮影し、自動的に1つのPDFファイルに結合するPythonスクリプトです。電子書籍やドキュメントのデジタルアーカイブ化を効率化することを目的としています。

## 2. 動作環境
- **OS**: Windows 10 / 11
- **言語**: Python 3.10以上
- **主要ライブラリ**:
  - `pyautogui`: マウス・キーボード操作およびスクリーンショット撮影
  - `Pillow (PIL)`: 画像処理、JPEG保存、PDF結合
  - `keyboard`: グローバルホットキー（ESC）の監視
  - `tkinter`: ユーザーインターフェース（領域選択、コントロールパネル）
  - `ctypes`: Windows API（ウィンドウ制御）の呼び出し

## 3. 主要機能

### 3.1 領域選択 UI
- 全画面を半透明（不透明度40%）のグレーで覆い、視覚的に選択範囲を明示。
- マウスドラッグにより矩形範囲（ピンク色の塗りつぶし）を指定。
- 選択完了後、指定された座標（x1, y1, x2, y2）をシステムに渡す。

### 3.2 フローティングコントロールパネル
- **UI特性**: 画面中央上部に表示。常に最前面（Topmost）かつ、不透明度90%で表示。
- **トグルボタン**: 動作状態に応じて「一時停止」と「再開」が切り替わる。
- **強制終了ボタン**: 途中でキャプチャを切り上げ、そこまでのデータでPDFを作成。
- **ステルス機能**: スクリーンショット撮影の瞬間だけパネルを非表示にし、画像への映り込みを防止。

### 3.3 自動キャプチャロジック
- **ページめくり**: `PageDown` キーを送信して自動でページを遷移。
- **重複検知（自動終了）**: 前後のスクリーンショットを比較（ImageChops.difference）し、5回連続で変化がない場合に「最終ページ」と判断して自動終了。
- **フォーカス管理**: コントロールパネル操作後にフォーカスが外れるのを防ぐため、Windows API（SetForegroundWindow）を使用して対象ウィンドウを自動的にアクティブ化。

### 3.4 PDF変換とファイル管理
- **画像保存**: `screenshots/` フォルダに一時的にJPEG形式で保存。
- **PDF結合**: 保存された全画像をA4相当のPDFに結合。
- **エラーハンドリング**: 既存のPDFが他アプリで開かれている（ロックされている）場合、ファイル名にタイムスタンプを付与して別名保存。

## 4. ウィンドウ管理の設計
本ツールはキャプチャ品質を保つために高度なウィンドウ制御を行っています。
- **ターミナルの隠蔽**: 実行中、ターミナルウィンドウは自動的に最小化され、かつウィンドウスタックの最背面（HWND_BOTTOM）に送られる。
- **クリックレス・フォーカス**: 対象ウィンドウをアクティブにする際、マウスクリックを使用せず `WindowFromPoint` と `SetForegroundWindow` を使用することで、意図しないテキスト選択（全選択状態）を防止。

## 6. 今後の課題 / 拡張性
- **ページめくりキーの変更**: 現在は `PageDown` 固定だが、設定ファイルから `Right` 等に変更可能にする。
- **待機時間の最適化**: ネットワーク環境やアプリの応答速度に合わせて、ページめくり待ち時間を動的に調整する機能。
- **GUI設定画面**: 起動時にPDFファイル名や保存場所をGUIで指定できる機能。
